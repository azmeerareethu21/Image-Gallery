# Responsive Image Gallery

## Description
This project is a responsive image gallery built using HTML5 and CSS3. It uses CSS Grid and Media Queries to create a responsive layout that displays three columns on desktop and one column on mobile devices.

## Features
- Responsive image gallery
- CSS Grid layout
- 3-column layout on desktop
- 1-column layout on mobile
- Hover zoom effect
- Clean and modern UI

## Technologies Used
- HTML5
- CSS3
- CSS Grid
- Media Queries
